# 6. Fibonacci is bad
Created Monday 06 January 2020


* For the 50th fibonacci using f(n) = f(n-1) + f(n-2). It takes 2^50^ = (1024)^5 ^= 10^15 ^steps.

We are better off with manual work in this case.
